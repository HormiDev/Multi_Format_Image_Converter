Quick manual View steps
  1. Drag one or more images into the input area or press Choose files.
  2. Select the output format in the export panel.
  3. Adjust format options such as resolution, quality, background, alpha, palette, or depth.
  4. Leave Keep original resolution enabled or disable it to change size.
  5. Leave Package as ZIP enabled if you want to download everything in a single file.
  6. Press Convert and wait for the download to finish.

The app runs locally. Native formats depend on the browser, and special formats use encoders included in the project.
